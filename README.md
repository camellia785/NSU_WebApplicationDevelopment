# NSU_WebApplicationDevelopment  
## Networking Fundamentals — Personal Website

**Course:** MIS 210 — Web Application Development (Fall 2025)  
**Author:** Yeona Choi  
**Tech Stack:** HTML · CSS · JavaScript (Vanilla)

A personal educational website that explains **networking fundamentals** with a clear focus on **backend development**.

---

## Project Overview

This project was built to:

- Strengthen core web development skills (HTML, CSS, JavaScript)
- Study networking concepts **from a backend engineer’s perspective**
- Turn abstract protocols into structured, readable explanations

The site presents networking topics in a **modal-based UI**, allowing users to focus on one concept at a time without losing context.

---

## Why Networking for Backend?

Backend systems always operate inside a network.

Understanding networking helps backend developers:

- Design correct HTTP APIs (methods, headers, status codes)
- Reason about latency, timeouts, and connection failures
- Implement authentication securely (cookies, sessions, JWT)
- Debug issues beyond application code (DNS, TLS, routing)

This project connects networking theory directly to real backend use cases.

---

## Site Structure

The website is organized into the following sections:

- **Home**  
  Project purpose, learning goals, and target audience

- **Basics**  
  Internet fundamentals: HTTP/HTTPS, DNS, domains, hosting, browsers

- **Deep Dive**  
  Core backend networking topics: TCP/UDP, handshakes, REST APIs, security

- **Practice**  
  Real-world backend architecture: load balancing, proxies, cloud networking

- **Contact**  
  Portfolio connection point (Email · GitHub · LinkedIn)
